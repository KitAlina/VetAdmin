# VetAdmin
Конфигурация «1С:Предприятие 8»

Проект для отработки практических навыков разработки в 1С.

## Структура конфигурации

### Подсистема «Сотрудникам»
* Документы с печатными формами
* Отчет «Прайс-лист»
* Отчеты по продажам с фильтрацией по критериям

### Подсистема «Общее»
* Документ «Расписание процедур» на базе планировщика заданий

### Подсистема «Справочная информация»
* Справочники
* Перечисления  
* Регистры сведений и накопления

---

**Техническая реализация:** Встроенный язык 1С, СКД для отчетов, механизм планировщика.
